Estructura de una página web

¿Qué es una página web?

Una página web es un archivo de texto que, en vez de en español, está escrito en HTML. A diferencia de otros lenguajes, este no se utiliza para programar algoritmos, sino con el fin de estructurar el contenido de una página web.

Por lo tanto, HTML no es un lenguaje de programación, sino un lenguaje de marcas. En lugar de variables, funciones, clases o estructuras de control, utiliza etiquetas que definen el rol de cada elemento en la página.
