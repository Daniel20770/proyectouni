Unidad 4

Estructura de una página web

Una página web es un archivo de texto que, en vez de en español, está escrito en HTML. A diferencia de otros lenguajes, este no se utiliza para programar algoritmos (desarrollo de código tradicional), sino con el ﬁn de estructurar el contenido de una página web. Por lo tanto, HTML no es un lenguaje de programación, sino de marcas, motivo por el que, en vez de variables, funciones, clases o estructuras de control, utiliza etiquetas (marcas) que establecen el papel que juega cada uno de los elementos que conforman la página.
