Estructura de una página web

¿Qué es una página web?

Una página web es un archivo de texto que, en vez de estar escrito en español, está redactado en HTML. A diferencia de otros lenguajes, este no se utiliza para programar algoritmos, sino para estructurar el contenido de una página web.

Por lo tanto, HTML no es un lenguaje de programación, sino un lenguaje de marcas. En lugar de variables, funciones o estructuras de control, se utilizan etiquetas que indican la función de cada elemento dentro de la página.
